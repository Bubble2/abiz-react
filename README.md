## 安装


> `generator-abiz-react-app`脚手架基于`yeoman`开发，所以需要先安装`yeoman`

### 全局安装`yeoman`和`generator-abiz-react-app`

```
npm install -g yo generator-abiz-react-app
```

### 创建项目目录

```
//创建项目文件夹
mkdir [projectName]

//进入项目文件夹
cd [projectName]
```

### 生成项目

```
yo abiz-react-app
```

### 启动项目

```
npm run dev
```
